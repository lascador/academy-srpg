# 🎮 Academy SRPG — 기획 문서

> **이 문서의 목적:** AI 어시스턴트(Copilot 등)가 새 세션에서 이 문서를 읽으면
> 프로젝트의 방향과 현재 상태를 즉시 파악할 수 있도록 정리한 기획 메모입니다.

## 0. 작업자 참고사항 (AI 어시스턴트가 반드시 읽을 것)

- **작업자는 프로그래밍/코딩 비전공자**입니다.
- VS Code에서 Copilot에게 자연어로 지시하는 **바이브코딩** 방식으로 개발 중입니다.
- 코드를 직접 읽고 수정하는 능력은 제한적입니다.

## 1. 한줄 요약

**파이어 엠블렘** 스타일의 턴제 SRPG 전투 + **프린세스 메이커 / 우마무스메** 스타일의 육성 시뮬레이션을 결합한 게임.

---

## 2. 세계관 & 배경

- **장르 톤:** 한국 웹소설풍 이세계 판타지 아카데미물
- **무대:** 제국에서 가장 이름 높은 교육기관(아카데미)
- **주인공:** 아카데미에 입학하며 이야기가 시작됨

---

## 3. 게임 루프 (핵심 흐름)

```
┌─────────────────────────────────────────────┐
│              아카데미 생활 (육성 파트)           │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │ 강의/훈련  │  │ 탐색      │  │ 교감/교류  │  │
│  │ → 능력치↑ │  │ → 이벤트  │  │ → 관계치↑ │  │
│  └──────────┘  └──────────┘  └──────────┘  │
│         주 단위로 행동 선택 → 스트레스 관리       │
└────────────────────┬────────────────────────┘
                     │ 일정 주기마다
                     ▼
┌─────────────────────────────────────────────┐
│            전투 이벤트 (SRPG 파트)             │
│  - 시험, 또는 예상치 못한 실전                   │
│  - 그리드 기반 턴제 전투                        │
│  - 패배 시 → 게임 오버                         │
└─────────────────────────────────────────────┘
```

## 4. 핵심 시스템 설계

### 4-1. 육성 파트 (`AcademyManager`)
| 요소 | 설명 |
|---|---|
| **주(week) 단위 진행** | 매주 활동 1개를 선택 |
| **활동 종류** | 훈련(능력치↑ / 스트레스↑), 교류(관계치↑), 휴식(스트레스↓) 등 |
| **능력치** | HP, 공격력, 방어력 — `CharacterStats` |
| **스트레스** | 0~100 범위, 누적되면 패널티 (상세 미정) |
| **활동 데이터** | `ActivityData` ScriptableObject로 정의 |

### 4-2. SRPG 전투 파트 (`BattleManager` + `TurnManager`)
| 요소 | 설명 |
|---|---|
| **그리드** | `GridManager` — 기본 8×8 타일맵, 타일별 이동 가능 여부 판정 |
| **턴 시스템** | 스피드 기반 쿨다운 방식 (`100 / speed`), 쿨다운이 0이 된 유닛부터 행동 |
| **유닛** | `Unit` — HP, 공격력, 방어력, 속도, 이동 범위, 공격 범위 |
| **전투 흐름** | 유닛 선택 → 이동 → 공격 → 턴 종료 |
| **승리/패배** | 아군 또는 적 전멸 시 전투 종료 |

### 4-3. UI (`AcademyUI`)
- 현재 주(week), 능력치, 스트레스 표시
- 훈련 / 교류 / 휴식 버튼으로 활동 실행

## 8. 프로젝트 구조

```
Academy-srpg/
├── Assets/
│   ├── Scripts/
│   │   ├── ActivityData.cs      ← 활동 데이터 (ScriptableObject)
│   │   ├── AcademyManager.cs    ← 육성 파트 매니저
│   │   ├── AcademyUI.cs         ← 육성 UI
│   │   ├── BattleManager.cs     ← 전투 진행 컨트롤러
│   │   ├── CharacterStats.cs    ← 캐릭터 능력치
│   │   ├── GridManager.cs       ← 그리드/타일맵 관리
│   │   ├── TurnManager.cs       ← 턴 순서 관리
│   │   └── Unit.cs              ← 전투 유닛
│   ├── TextMesh Pro/            ← TMP 기본 에셋
│   └── InputSystem_Actions.inputactions
└── ProjectSettings/
```
